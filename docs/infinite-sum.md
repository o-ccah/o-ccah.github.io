# 無限和のノート

[無限和のノート](files/infinite-sum-20200324.pdf)

## 概要

無限和について，できるだけ一般的なセッティングでまとめました．実は「級数」という語が一度も出てきません．そのうち無限積のことを書き足したいと思っています．
